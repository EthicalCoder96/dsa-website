Linked List

A linked list is a linear data structure where elements are stored in nodes, and each node points to the next node in the sequence. It provides dynamic memory allocation and efficient insertion and deletion operations.

Searching

Searching is the process of finding the location or occurrence of a particular item (or key) in a collection of data, such as an array, list, or database. The objective is to determine whether the item is present and, if so, to identify its position within the data structure.

Sorting

Sorting is the process of arranging a collection of items in a specific order, often ascending or descending based on some criteria. The objective is to organize the data in a way that facilitates efficient retrieval, searching, and other operations.

Stack

A stack is a linear data structure that follows the Last In, First Out (LIFO) principle. Elements are added and removed from the same end, known as the top. It supports operations such as push (to add an element) and pop (to remove the top element).

Queue

A queue is a linear data structure that follows the First In, First Out (FIFO) principle. Elements are added at the rear and removed from the front. It supports operations such as enqueue (to add an element) and dequeue (to remove the front element).

Trees

A tree is a hierarchical data structure consisting of nodes connected by edges. The topmost node is called the root, and each node can have zero or more child nodes. Nodes with no children are called leaves.
